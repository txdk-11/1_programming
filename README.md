비밀번호 강도 분석 및 생성 웹사이트

📌 개발 목표

이 프로젝트는 사용자가 입력한 비밀번호의 강도를 실시간으로 분석하고, 안전한 비밀번호를 생성할 수 있는 웹사이트입니다.

주요 기능:
- ✅ 사용자가 입력한 비밀번호의 강도 실시간 분석
- ✅ 보안 점수 및 등급 표시 (0~100점, 4단계)
- ✅ 취약한 부분 안내 및 개선 방법 제시
- ✅ 안전한 비밀번호 생성
- ✅ 보안 상식 및 수칙 제공

---

🛠️ 사용 기술

- **HTML5** : 웹페이지 구성 및 구조화
- **CSS3** : 반응형 디자인 및 인터랙티브 스타일링
- **JavaScript (ES6+)** : 비밀번호 강도 분석 알고리즘 및 생성 기능

🎯 주요 기능

### 1️⃣ 비밀번호 강도 분석

#### 실시간 분석
- 사용자가 입력할 때마다 자동으로 분석
- 입력 완료 후 피드백 즉시 제공

#### 분석 항목 (7가지)
| 항목 | 설명 | 배점 |
|------|------|------|
| 길이 | 8자 이상 | 20점 |
| 대문자 | A-Z 포함 여부 | 20점 |
| 소문자 | a-z 포함 여부 | 15점 |
| 숫자 | 0-9 포함 여부 | 15점 |
| 특수문자 | !@#$% 등 포함 여부 | 20점 |
| 반복 문자 | 같은 문자 3회 이상 연속 | ±10점 |
| 쉬운 패턴 | 123456, password 등 | ±20점 |

#### 보안 점수 및 등급
```
0~39점   → 매우 약함 ⚠️ (빨강)
40~59점  → 약함 ⚠️ (주황)
60~79점  → 보통 👍 (노랑)
80~100점 → 강함 ✓ (초록)
```

### 2️⃣ 강도 게이지
- 색상 막대로 보안 수준 시각적 표시
- 점수에 따른 자동 변색
- 부드러운 애니메이션 효과

### 3️⃣ 취약점 안내
사용자 맞춤형 피드백 제공:
```
예시:
- "비밀번호 길이: 현재 5자 (최소 8자 권장)"
- "대문자(A-Z)를 추가하면 더 강합니다."
- "숫자(0-9)를 추가하세요."
- "특수문자(!@#$% 등)를 포함하면 훨씬 더 안전합니다."
- "같은 문자가 반복됩니다. 다른 문자를 사용하세요."
- "너무 쉬운 패턴(123456, qwerty 등)입니다. 피하세요."
```

### 4️⃣ 분석 항목 체크리스트
실시간으로 충족 여부 표시:
- ✓ 길이 8자 이상
- ✓ 대문자 포함
- ✓ 소문자 포함
- ✓ 숫자 포함
- ✓ 특수문자 포함
- ✓ 반복 문자 없음
- ✓ 쉬운 패턴 없음

### 5️⃣ 안전한 비밀번호 생성

#### 사용자 정의 옵션
- 🎯 길이 선택 (8~32자, 슬라이더 또는 입력)
- 🔤 포함할 문자 선택:
  - 대문자 (A-Z)
  - 소문자 (a-z)
  - 숫자 (0-9)
  - 특수문자 (!@#$%^&* 등)

#### 생성된 비밀번호 예시
```
Q7#mLp2!Az9&
xK9$bP2nQw#L
5Ry@2jH8vM3p
```

### 6️⃣ 복사 버튼
- 생성한 비밀번호를 클릭 한 번으로 클립보드 복사
- 복사 완료 시 버튼 상태 변경으로 피드백

### 7️⃣ 보안 상식 (6가지)

#### 1. 고유한 비밀번호 사용
같은 비밀번호를 여러 사이트에서 사용하지 마세요. 한 사이트가 해킹되면 다른 모든 계정이 위험해집니다.

#### 2. 주기적으로 변경하기
최소 3개월마다 비밀번호를 변경하세요. 특히 중요한 계정은 더 자주 변경하는 것이 좋습니다.

#### 3. 2단계 인증 사용하기
가능한 모든 서비스에서 2단계 인증(2FA)을 활성화하세요. 이는 비밀번호 만으로는 부족한 추가 보안을 제공합니다.

#### 4. 개인 정보 포함 금지
생년월일, 이름, 전화번호 등 개인정보를 비밀번호에 포함하지 마세요.

#### 5. 공공 컴퓨터에서 주의
카페나 도서관의 공공 컴퓨터에서는 중요한 계정에 로그인하지 마세요.

#### 6. 비밀번호 관리자 사용
LastPass, 1Password 등의 비밀번호 관리자를 사용하여 안전하게 보관하세요.

🧮 핵심 알고리즘

### 점수 계산 알고리즘

```javascript
score = 0

1. 길이 확인 (8자 이상: +20점)
2. 대문자 포함 검사 (+20점)
3. 소문자 포함 검사 (+15점)
4. 숫자 포함 검사 (+15점)
5. 특수문자 포함 검사 (+20점)
6. 반복 문자 확인 (+10점 또는 -10점)
7. 쉬운 패턴 확인 (+10점 또는 -20점)

최종 점수 = min(100, max(0, score))

등급 결정:
- 0~39점: 매우 약함
- 40~59점: 약함
- 60~79점: 보통
- 80~100점: 강함
```

### 패턴 인식

#### 반복 문자 감지
정규식: `(.)\1{2,}` (같은 문자 3회 이상 연속)
```
예: aaa, 111, SSS → 감지됨
```

#### 쉬운 패턴 감지
다음 패턴들을 포함하면 감지:
- 숫자 시퀀스: 123456, 12345678, 1234567890
- 반복 숫자: 111111, 000000, 999999
- 일반적인 단어: password, admin, test, demo
- 키보드 시퀀스: qwerty, asdfgh, zxcvbn
